## RECORDING TIPS

1. **Use a good microphone** — Voice clarity matters
2. **Show PTT indicator** — So viewers see when you're speaking
3. **Natural pacing** — Not too fast, not robotic
4. **Show real demos** — Live is more convincing than slides
5. **Have backup** — Pre-record segments if possible
