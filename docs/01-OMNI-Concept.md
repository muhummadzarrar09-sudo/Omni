# OMNI — Autonomous Personal Agent

**Hackathon Entry:** Agentic AI Innovation Challenge 2026 — Open Innovation Category

---

## Vision Statement

> *OMNI is the accessibility-first autonomous agent that bridges human intention and digital action—enabling anyone, including people without hands, to navigate computers through voice.*

---

## The Problem

1. **Accessibility Gap**: 85% of computing tasks require hands/mouse
2. **AI Agent Fragmentation**: Current agents require cloud, are always-listening (privacy risk)
3. **Solo Developer Reality**: Building a complete agent requires stitching together 10+ tools

---

## OMNI's Solution: Option C

**Accessibility-first design that everyone loves:**

| User Type | How They Use OMNI |
|-----------|-------------------|
| **Hands-free user** | Primary use case — full voice control |
| **Developer** | Voice-navigate code, run terminal commands |
| **Productivity user** | Voice email, browser automation |
| **Power user** | Custom command shortcuts, workflow automation |

---

## Core Features

- 🎤 **Push-to-Talk** — CapsLock to speak, privacy-safe
- 🌐 **Browser Control** — CDP via Chrome DevTools Protocol
- 💻 **Windows Automation** — UIA-based window control
- ⌨️ **VS Code Integration** — WebSocket bridge
- 🔊 **Local TTS** — Kokoro TTS for instant feedback
- 🔒 **Privacy-First** — All processing local
