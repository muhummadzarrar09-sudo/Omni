## SLIDE DESIGN NOTES

**Color Scheme**:
- Background: `#1a1a2e` (Dark navy)
- Primary: `#00d9ff` (Cyan)
- Text: `#ffffff` (White)
- Secondary: `#8892b0` (Muted gray)

**Typography**:
- Headers: Bold sans-serif, 48pt
- Body: Regular sans-serif, 24pt
- Code/Commands: Monospace, 20pt

**Layout**:
- Center-aligned content
- Plenty of whitespace
- Icons for visual interest
- Consistent padding

**Tools**:
- Can use Canva, Google Slides, or PowerPoint
- Export as PDF for consistency
